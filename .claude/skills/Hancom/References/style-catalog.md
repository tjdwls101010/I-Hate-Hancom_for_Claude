# 기반 템플릿 스타일 카탈로그

## 테두리/배경 스타일 (borderFill)

표 셀에서 `<hp:tc borderFillIDRef="N">`으로, 문단에서 `<hh:border borderFillIDRef="N">`으로 참조한다.

| ID | 테두리 | 배경색 | 용도 |
|----|--------|--------|------|
| 1 | 없음 | 없음 | **기본값** — 일반 문단 |
| 2 | 사방 실선 0.12mm | 없음 | 기본 표 셀 |
| 3 | 없음 | 없음 | 문단용 (별도 구분) |
| 4 | 사방 실선 0.12mm | 회색 #D9D9D9 | **표 헤더** (가장 일반적) |
| 5 | 사방 실선 0.12mm | 연파랑 #DFE6F7 | 표 헤더 대안 |
| 6 | 사방 실선 0.12mm | 흰색 #FFFFFF | 표 본문 셀 (명시적 흰 배경) |
| 7 | 없음 | 흰색 #FFFFFF | 배경 흰색만 |
| 8 | 상하만 실선 0.12mm | 없음 | 수평 구분선 표현 |

## 일반적인 조합 패턴

### 보도자료 제목 영역
```xml
<hp:p paraPrIDRef="9" styleIDRef="0">
  <hp:run charPrIDRef="6"><hp:t>2026년 소상공인 지원 대책</hp:t></hp:run>
</hp:p>
```
→ paraPr 9(가운데, 상하여백) + charPr 6(24pt HY헤드라인M 굵게)

### 본문 □ 소제목
```xml
<hp:p paraPrIDRef="0" styleIDRef="0">
  <hp:run charPrIDRef="3"><hp:t>□ 주요 추진 내용</hp:t></hp:run>
</hp:p>
```
→ paraPr 0(양쪽정렬) + charPr 3(14pt 바탕 굵게)

### 본문 ㅇ 설명
```xml
<hp:p paraPrIDRef="7" styleIDRef="0">
  <hp:run charPrIDRef="1"><hp:t>ㅇ 소상공인 대출 금리를 인하한다.</hp:t></hp:run>
</hp:p>
```
→ paraPr 7(들여쓰기) + charPr 1(12pt 바탕)

### 본문 - 세부항목
```xml
<hp:p paraPrIDRef="14" styleIDRef="0">
  <hp:run charPrIDRef="1"><hp:t>- 대상: 연매출 3억 원 이하</hp:t></hp:run>
</hp:p>
```
→ paraPr 14(이중 들여쓰기) + charPr 1(12pt 바탕)

### 표 헤더 셀
```xml
<hp:tc borderFillIDRef="4" width="14000">
  <hp:subList>
    <hp:p paraPrIDRef="4" styleIDRef="0">
      <hp:run charPrIDRef="7"><hp:t>구분</hp:t></hp:run>
    </hp:p>
  </hp:subList>
</hp:tc>
```
→ borderFill 4(회색 배경) + paraPr 4(가운데/가운데) + charPr 7(10pt 굵게)

### 표 본문 셀
```xml
<hp:tc borderFillIDRef="2" width="14000">
  <hp:subList>
    <hp:p paraPrIDRef="5" styleIDRef="0">
      <hp:run charPrIDRef="8"><hp:t>500억 원</hp:t></hp:run>
    </hp:p>
  </hp:subList>
</hp:tc>
```
→ borderFill 2(실선만) + paraPr 5(왼쪽/가운데) + charPr 8(10pt)
