## 2. 최적 토픽 수 선택 과정

- **기법**: Elbow Method (설명 분산 분석)
- **토픽 범위**: 5~30개
- **Elbow Point**: 26개 토픽 (설명 분산: 0.1567)

### 2.2 최종 선택
- **선택 토픽 수**: 10개
- **총 설명 분산**: 0.0935

- **선택 근거**:
  1. K-means 4개 군집의 2~3배 고려
  2. Cluster 2 (43%)의 과도한 크기 세분화 필요
  3. 비즈니스 해석 가능성
  4. Phase 4.2 NMF와의 비교 용이성

## 10. Phase 4.2 (NMF)를 위한 권장사항

### 10.1 LSI 기준선 설정
- **토픽 수**: 10개 (NMF도 동일하게 설정하여 직접 비교)
- **설명 분산**: 0.0935
- **일치도 기준**: ARI 0.2333, NMI 0.3735

### 10.2 비교 포인트
1. **해석 가능성**: LSI는 음수 가중치 허용, NMF는 비음수 제약
2. **토픽 품질**: 키워드 일관성, 의미 명확성
3. **문서-토픽 분포**: 희소성(sparsity) 차이
4. **군집 일치도**: K-means와의 ARI/NMI 비교
5. **계산 효율**: 학습 시간 및 메모리 사용량

### 10.3 평가 기준

<!-- box:info -->
- **정량적**: Reconstruction Error, Topic Coherence, Sparsity
- **정성적**: 토픽 레이블의 명확성, 키워드 해석 가능성, 비즈니스 의미 도출 용이성
<!-- /box -->

### 10.4 기대 결과
- LSI: 전반적 의미 구조 파악, 음수 가능 (반대 개념)
- NMF: 명확한 부분 기반 표현, 해석 용이성
- 두 방법의 교집합 → 신뢰도 높은 토픽
- 차이점 → 각 방법의 고유 발견
